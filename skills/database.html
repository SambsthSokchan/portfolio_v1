Understanding Database & Cloud Technologies

SQL Databases: Standardized Relational Storage

PostgreSQL & MySQL

PostgreSQL and MySQL are the world's most popular open-source relational database management systems. They store data in highly structured tables (like advanced spreadsheets) with strict rules defining how different arrays of data connect to each other.

SQLite

SQLite is a C-language library that implements a small, fast, self-contained, high-reliability SQL database engine. Instead of running on a heavy server connection, it runs directly inside a unified application file, making it perfect for local testing or mobile apps.

NoSQL: Flexible Document Storage

MongoDB is a leading NoSQL database. Instead of structured tables, it stores data in flexible, JSON-like documents. Because fields can vary from document to document, MongoDB is highly advantageous for rapidly evolving projects where the data structure isn't completely finalized.

Tooling & Environments

XAMPP

XAMPP is an easy-to-install local server environment containing Apache, MariaDB/MySQL, and PHP. It's the classic, hassle-free way to simulate a real web server on your personal computer for testing before going live.

SSMS

SSMS (SQL Server Management Studio) is a powerful graphical interface provided by Microsoft for configuring, managing, and administering components of Microsoft SQL Server. It allows developers to visually build and manage databases instead of relying purely on a raw command line.

Cloud & Containerization

AWS (Amazon Web Services)

AWS is the world's most comprehensive cloud platform. Instead of buying physical hardware, you rent incredibly powerful servers, databases, and network architecture on-demand from Amazon's global data centers to scale your applications infinitely.

Docker

Docker allows you to package an application alongside all of its dependencies into a single digital "container". This completely eliminates the "it works on my machine" problem, guaranteeing that your application will run identically whether executing on your laptop or deep inside the server farms on AWS.
